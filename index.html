РУКОВОДСТВО ПОДКЛЮЧЕНИЯ

01. Импорт Пресета

Скачайте JSON файл или скопируйте основной промпт из раздела "Промпты".

Перейдите в SillyTavern -> Extensions -> ...

02. Настройка Context Template

Убедитесь, что порядок инъекций (Injection Order) выставлен верно.

03. Активация Ренетт

После установки всех модулей, напишите в чат команду активации или просто начните ролеплей.
